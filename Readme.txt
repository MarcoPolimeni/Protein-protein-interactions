Installation

In order to be able to run this notebook, please follow these instruction:

1) Install Anaconda (or Miniconda to save space) according with you operating system (OS) https://docs.anaconda.com/anaconda/install/

2 )Install faunus https://faunus.readthedocs.io/en/latest/_docs/install.html

3) Create and activate a conda environment which contains all the packages needed.
   
	- conda env create -f environment.yml
        - conda activate python_project

4) Run the jupyter notebook project.ipynb
